# E-Commerce API

## Project Description

A simple REST API built with Node.js, Express.js, and MongoDB for managing products and shopping carts.

## Features

- Product CRUD operations
- Create Cart
- Get Cart
- Add Items to Cart
- MongoDB Integration
- Central Error Handling

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

Install dependencies:

```bash
npm install
```

## Run the Project

```bash
npm start
```

or

```bash
npm run dev
```

## API Endpoints

### Products

- GET /api/products
- POST /api/products
- PUT /api/products/:id
- DELETE /api/products/:id

### Cart

- POST /api/cart
- GET /api/cart/:id
- POST /api/cart/:id/items
